What is thead on the table?

The thead in a table defines the header section, grouping the column headings separately from the table body. It helps to format and organize table headers.

What is tr?

The tr element in HTML represents a table row. It is used to define a row within a table and contains one or more table cells

What is th?

The th element in HTML represents a table header cell. It is used to define the heading for a column or row in a table and is typically bold and centered by default. It helps identify the type of data contained in the corresponding columns or rows.

What is td?

The td element in HTML represents a table data cell. It is used to define a standard cell in a table that contains data or content. Each td element holds the actual data within a row tr of a table.

Can we use block elements in inline elements?

No, block-level elements should generally not be placed inside inline elements, as it violates HTML structure rules and can lead to unpredictable rendering behavior in browsers.

What are difference between absolute vs relative paths?

Absolute paths provide the full URL or file location starting from the root, while relative paths specify the location relative to the current directory or document.
